# Умный дом (Smart Home)

### Тема
**Умный дом (Smart Home):** управление устройствами по комнатам, события от устройств и правила автоматизации (реакция на события — включить свет, уведомить пользователя и т.д.). Удаление комнаты запрещено, если в ней есть устройства или привязанные правила.


| Сущность | Описание |
|----------|----------|
| **Room** | Комната (название, описание). В комнате находятся устройства. |
| **Device** | Устройство (название, тип: LAMP, SENSOR и т.д., привязка к комнате, активность). Генерирует события. |
| **User** | Пользователь (имя, email). Может быть получателем уведомлений в правилах автоматизации. |
| **Event** | Событие от устройства (тип события, payload, время). Правила автоматизации реагируют на тип события. |
| **AutomationRule** | Правило автоматизации: тип события-триггера, тип действия (включить устройство, уведомить пользователя), привязка к комнате/устройству/пользователю, активность. |

### CRUD по сущностям

| Сущность | Эндпоинты |
|----------|------------|
| Room | `POST/GET /api/rooms`, `GET/PUT/DELETE /api/rooms/{id}` |
| Device | `POST/GET /api/devices`, `GET/PUT/DELETE /api/devices/{id}` (фильтр `?roomId=...`) |
| User | `POST/GET /api/users`, `GET/PUT/DELETE /api/users/{id}` |
| Event | `POST/GET /api/events`, `GET/PUT/DELETE /api/events/{id}` (фильтр `?deviceId=...`) |
| AutomationRule | `POST/GET /api/automation-rules`, `GET/PUT/DELETE /api/automation-rules/{id}` |

Удаление комнаты: **409 Conflict**, если в комнате есть устройства или правила.

### Бизнес-операции (Задание 3)

| Сценарий | Метод | URL | Тело запроса (JSON) |
|----------|--------|-----|---------------------|
| Движение -> включить свет | POST | `/api/scenarios/motion-detected` | `{"deviceId": 4}` |
| Нет движения 30 сек -> выключить свет | POST | `/api/scenarios/no-motion-timeout` | `{"deviceId": 4, "elapsedSeconds": 30}` |
| Протечка -> уведомление пользователю | POST | `/api/scenarios/leak-detected` | `{"deviceId": 5}` |
| Колонка -> включить свет в комнате | POST | `/api/scenarios/speaker-turn-on` | `{"deviceId": 6, "roomName": "Санузел"}` |
| Колонка -> выключить свет в комнате | POST | `/api/scenarios/speaker-turn-off` | `{"deviceId": 6, "roomName": "Санузел"}` |

Журнал событий для демонстрации:
`GET /api/events/log` (устройство + комната + событие + что было сделано).
